Proyectos Destacados

Around The U.S.

Contexto: Aplicación web interactiva desarrollada durante el bootcamp de TripleTen (Sprints 8-12). El proyecto evolucionó desde una galería estática a una aplicación completa con integración de API.

Descripción: Plataforma que permite a usuarios gestionar su perfil y compartir fotos de lugares alrededor de los Estados Unidos. Los usuarios pueden editar su información, agregar nuevas fotos, dar "me gusta" y eliminar tarjetas.

Análisis y aprendizajes:
  • Implementé validación de formularios HTML5 con manejo avanzado de errores
  • Refactoricé el código aplicando Programación Orientada a Objetos (POO) con clases reutilizables
  • Integré peticiones Fetch API para conectar con un servidor backend
  • Manejé el ciclo de vida de componentes y estado de la aplicación
  • Creé módulos ES6 para mejor organización del código
Tecnologías: HTML5 CSS3 (BEM) JavaScript ES6+ Fetch API POO
Conclusiones: Este proyecto fue fundamental para entender la importancia de la arquitectura de código y cómo estructurar aplicaciones escalables. La progresión de 4 sprints me permitió ver cómo una aplicación puede mejorarse iterativamente.

Cafetería Triple Peaks

Contexto: Proyecto de sitio web para una cafetería de especialidad. Desarrollado como parte de las prácticas de frontend durante el bootcamp, enfocándose en diseño responsivo y buenas prácticas HTML/CSS.

Descripción: Página web profesional para "Triple Peaks Coffee Shop" que incluye información de horarios, galería de recetas de café con videos instructivos, y un sistema de reserva de mesas con formulario interactivo.

Análisis y aprendizajes:
  • Diseñé una estructura HTML5 semántica y accesible
  • Aplicué la metodología BEM para CSS organizado y mantenible
  • Implementé Flexbox para layouts responsivos sin dependencias externas
  • Integré videos de YouTube de forma embebida
  • Creé formularios con validación básica HTML5
  • Trabajé con Google Fonts para tipografía moderna
Tecnologías: HTML5 CSS3 (BEM) Flexbox Google Fonts Responsive Design
Conclusiones: Este proyecto me enseñó la importancia del CSS bien organizado y cómo crear interfaces atractivas sin JavaScript. Reforzó mi comprensión de diseño responsivo y accesibilidad web.
